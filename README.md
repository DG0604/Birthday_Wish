# 🎂 Interactive Birthday Surprise Web App

A personalized, interactive single-page web application created as a unique birthday surprise. It features a fun login gate, soothing sunflower themes, dynamic confetti, and a beautiful ombre tulip celebration screen to help celebrate across the distance.

## ✨ Features

* **Interactive Login Gate:** Requires a specific username and a secret password with a helpful hint to unlock the birthday surprise.
* **Themed UI Design:** * *Login Screen:* Soothing soft sage green background decorated with sunflowers.
    * *Surprise Screen:* A beautiful baby pink to yellow ombre gradient with floating tulips.
* **Dynamic Animations:** Features custom CSS keyframe animations for floating elements and a full-screen confetti popper effect powered by JavaScript.
* **Responsive:** Simple and elegant layout that works seamlessly across desktop and mobile devices.

## 🛠️ Technologies Used

* **HTML5:** Structure and content.
* **CSS3:** Styling, gradients, flexbox layouts, and keyframe animations.
* **JavaScript (Vanilla):** DOM manipulation, login validation logic, and triggering the confetti library.
* **[Canvas Confetti](https://www.kirilv.com/canvas-confetti/):** A lightweight JS library for the party popper effects.

## 🚀 How to Run Locally

1. Clone this repository or download the ZIP file.
2. Extract the files to a folder on your computer.
3. Double-click the `index.html` file to open it directly in any modern web browser (Chrome, Safari, Firefox, Edge).
4. Enter the credentials to see the magic!
